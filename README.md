# How to Run

## Prerequisites

### Installing JDK

To run this game, you need Java and the Java compiler (javac) installed on your system. These come with the Java Development Kit (JDK).

- **Download and install JDK:**
  - Visit [Oracle's JDK download page](https://www.oracle.com/java/technologies/javase-downloads.html) and download the latest JDK.
  - Follow the installation instructions for your operating system.

- **Alternative installation methods:**
  - **Windows (using Chocolatey):** `choco install openjdk`
  - **macOS (using Homebrew):** `brew install openjdk`
  - **Linux (Ubuntu/Debian):** `sudo apt install openjdk-17-jdk`

After installation, verify by running `java -version` and `javac -version` in your terminal.

## Running the Game

1. Clone or download this repository.
2. Navigate to the `src` directory: `cd src`
3. Compile the game: `javac Game.java`
4. Run the game: `java Game`
